# Perfume Backend Project Memory

해당 작업에서는 작업 중 사용자의 지적 사항 및 반복하지 말아야 할 실수를 기록하는 문서입니다.
에이전트는 시작 전 **반드시** 해당 문서를 읽고, 작업 중 사용자가 작업 방식 또는 검증 방식에 대한 지적사항이 존재할 경우 사용자에게 영속적으로 저장할지 여부를 묻고, 요구시 이곳에 추가해야합니다.

내용은 다음과 같이 기록하세요.
 - 문제 제기 사항. 사용자가 어떤 부분에 대해 지적했는지에 대해서 작성하고, 해당 상황을 묘사합니다.
 - 해당 문제를 해결하기 위한 방안을 단계별로 작성하고, 사용자에게 평가를 받으세요.
 - 제시된 해결 방안의 검수를 통과 할 경우, 사용자에게 재발 방지를 위한 내용 작성에 여부에 대해 물어보고, 작성 요구시 여기에 작성하세요.
